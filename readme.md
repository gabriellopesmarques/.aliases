# aliases.sh

## install

### clone repository
```
cd ~
git clone <repo>
```

### using only aliases

#### zsh
```
echo "source ~/.aliases/aliases.sh" >> ~/.zshrc
```

#### bash
```
echo "source ~/.aliases/aliases.sh" >> ~/.bashrc
```


### all env

#### the great packages installer **curl**
sudo apt-get install curl -y

./i3/install.sh
