# aliases.sh

## install

### clone repository
```
cd ~
git clone <repo>
```

### using only aliases

#### zsh
```
echo "source ~/.aliases/aliases.sh" >> ~/.zshrc
```

#### bash
```
echo "source ~/.aliases/aliases.sh" >> ~/.bashrc
```


### all env

#### i3wm
```
./i3/install.sh
```

#### zsh
```
./oh-my-zsh/install.sh
```

#### tmux
```
./tmux/install.sh
```

#### vim
```
./vim/install.sh
```
